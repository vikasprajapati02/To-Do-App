# To-Do-App
A simple and responsive To-Do Application built using HTML, CSS, and JavaScript. This project focuses on core JavaScript concepts such as DOM manipulation, event handling, and Local Storage. Users can add, delete, and mark tasks as completed, and the data remains saved even after refreshing the page. 

A simple, clean, and responsive To-Do Application built using Vanilla JavaScript, designed to practice core frontend concepts such as DOM manipulation, event handling, and browser storage management.
This project helps users add, delete, and mark tasks as complete, with all tasks saved automatically using Local Storage.
